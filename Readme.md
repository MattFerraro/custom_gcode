This is an experimental art project to paint a portrait using a CNC router and a V-bit to punch holes

# To Use

```bash
git clone git@github.com:MattFerraro/custom_gcode.git
cd custom_gcode
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt

python mosaic.py
```
